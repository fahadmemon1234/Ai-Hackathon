# Hardware Setup

Choosing the right hardware is a crucial first step in your robotics journey. This page provides a guide to hardware setups for different budgets and goals.

## Recommended Hardware

| Category | Good (Cost-Effective) | Better (Balanced) | Best (High-Performance) |
|---|---|---|---|
| **Computer** | Raspberry Pi 4 | NVIDIA Jetson Orin Nano | NVIDIA Jetson AGX Orin |
| **Accelerator** | Google Coral USB Accelerator | (Integrated in Jetson) | (Integrated in Jetson) |
| **Microcontroller** | Arduino Uno / Raspberry Pi Pico | Teensy 4.1 | - |
| **Sensors** | Basic sensor kit (ultrasonic, IR) | Intel RealSense Depth Camera | Ouster LiDAR / ZED 2i Stereo Camera |
| **Actuators** | Standard Servo Motors | Dynamixel Servos | Brushless DC Motors with Encoders |

## Notes

- The **Good** setup is suitable for beginners and for learning the basics of robotics and ROS 2.
- The **Better** setup provides a significant performance boost and is recommended for more serious hobbyists and for running more complex AI models.
- The **Best** setup is for advanced users and researchers who need the highest level of performance for tasks like real-time 3D perception and manipulation.
